# 🗺️ 안전 도보 네비게이션 앱

> 장애인을 위한 범죄 위험 구역 회피 도보 네비게이션

---

## 소개

장애인이 안전하게 이동할 수 있도록 범죄 위험 구역을 분석하고,
이를 회피하는 최적 경로를 제시하는 도보 네비게이션 앱입니다.

한남대학교 컴퓨터공학과 졸업작품으로 진행된 팀 프로젝트이며,
학술제에서 **최우수상**을 수상했습니다.

---

## 담당 역할

- Tmap API 연동 및 네비게이션 기능 구현
- Android UI/UX 설계 및 개발

---

## 사용 기술

`Android` `Java` `Tmap API` `Android Studio`

---

## 수상
